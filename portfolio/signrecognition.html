Sign recognition

During the fall of 2014, I was writing a scientific paper. The purpose of the paper was to analyze the ability to read hand signs from the American sign language alphabet with a Leap motion sensor. The computer program I created used the output generated from the sensor and classified it by comparing the output data to data that were used to train the computer program. Comparing the data from the hand signs is performed using the k-nearest-neighbors algorithm (kNN).
